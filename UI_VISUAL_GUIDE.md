## Professional UI Elements

### Typography Hierarchy
```
H1: Page Title (centered, 2.5em, blue)
H2: Section Headers (2em, dark gray)
H3: Subsection Headers (1.5em, medium gray)
Body: Regular text (1em, black)
Caption: Small text (0.8em, light gray)
```

### Spacing System
```
Large gap: 40px (between major sections)
Medium gap: 20px (between subsections)
Small gap: 10px (between related items)
Tight: 5px (within components)
```

### Component Styles
```
Cards: Rounded corners (10px), shadow, padding
Buttons: Primary (blue), rounded, hover effects
Inputs: Gray border, white background, focus highlight
Expanders: Arrow icon, smooth animation
Metrics: Large number, small label, icon
```
